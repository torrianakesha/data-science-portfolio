## How to Get Started?
To explore or run any project locally:

### 1. Clone This Repository
```bash
git clone https://github.com/torrianakesha/data-science-portfolio.git
cd data-science-portfolio
```

### 2. Choose a Project
Navigate into one of the folders. Example:
```bash
cd machine-learning/training-deep-learning-model
```

### 3. Install Dependencies (if applicable)
```bash
pip install -r requirements.txt
```

### 4. Run / View 
Follow the README in each project’s folder to:
- Run notebooks
- Train models
- View dashboards (Power BI reports or visualizations)
